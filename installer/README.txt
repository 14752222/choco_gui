╚══════════════════════════════════════════════╝

【安装方式一】推荐 - 双击安装（VBS）

  双击 install.vbs → 自动请求管理员权限 → 选择安装位置 → 完成

【安装方式二】批处理安装

  右键 安装ChocoGUI.bat → 以管理员身份运行 → 按提示操作

【安装方式三】PowerShell 安装

  右键 setup.ps1 → 使用 PowerShell 运行

════════════════════════════════════════════════

安装完成后：
  - 程序位置: %LOCALAPPDATA%\Programs\ChocoGUI
  - 开始菜单: Choco GUI
  - 卸载方式: 开始菜单 → 卸载 Choco GUI

════════════════════════════════════════════════

系统要求：
  - Windows 10/11 (64位)
  - Chocolatey 包管理器 (https://chocolatey.org/install)
