# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅        |
| < 1.0   | ❌        |

## Reporting a Vulnerability

**Please do not file a public GitHub issue for security vulnerabilities.**

Instead, open a [GitHub Security Advisory](https://github.com/14752222/choco_gui/security/advisories/new) or send a direct message to the repository owner.

We aim to respond within **72 hours** and will coordinate a fix + disclosure timeline with you.

Thank you for helping keep Choco GUI safe! 🔐
